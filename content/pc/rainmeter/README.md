# Rainmeter Skins

Drop Rainmeter skin screenshots here. 

## File types accepted
- `.jpg` / `.png` — Preview screenshot of the skin on a desktop
- `.rmskin` — Actual installable skin (optional, add download link in metadata)

## Optional metadata

```json
{
  "title": "HUD Interface",
  "tags": ["dark", "hud", "minimal", "system"],
  "colors": ["dark"],
  "downloadUrl": "https://deviantart.com/...",
  "featured": false
}
```
