# Mobile Wallpapers

Drop your mobile wallpaper images here.

## Supported formats
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.avif`

## Optional metadata

Create a `.json` file with the same name as your image to add metadata:

```json
{
  "title": "My Wallpaper Title",
  "tags": ["minimal", "dark", "abstract"],
  "colors": ["dark", "minimal"],
  "featured": false
}
```

## Recommended resolution
- **Minimum:** 1080 × 1920px
- **Ideal:** 1290 × 2796px (iPhone 15 Pro Max)
